## Создание иконок:

Для полноценной работы PWA нужны иконки в формате PNG:

### Вариант 1: Использовать онлайн генератор
1. Зайдите на https://realfavicongenerator.net/
2. Загрузите вашу иконку (минимум 512x512px)
3. Скачайте сгенерированные иконки
4. Поместите `icon-192.png` и `icon-512.png` в папку `public/`

### Вариант 2: Создать вручную
Используйте любой графический редактор (Photoshop, GIMP, Figma):
- Создайте иконку 192x192px → сохраните как `public/icon-192.png`
- Создайте иконку 512x512px → сохраните как `public/icon-512.png`

## Как это работает:

### На мобильных устройствах:
1. При первом посещении сайта появится предложение установить приложение
2. Пользователь может установить через браузерное меню или кнопку "Установить"
3. После установки приложение будет работать как нативное:
   - Иконка на главном экране
   - Запуск в полноэкранном режиме
   - Работа офлайн (базовые функции)
   - Быстрый доступ

### На компьютерах:
- Сайт работает как обычный веб-сайт
- PWA функции доступны, но менее заметны
- Можно установить через браузер (Chrome, Edge)

## Тестирование:

### Локально:
```bash
npm run build
npm run preview
```

Откройте в браузере и проверьте:
- В DevTools → Application → Manifest (должен быть виден манифест)
- В DevTools → Application → Service Workers (должен быть зарегистрирован SW)

### На мобильном:
1. Задеплойте на Vercel
2. Откройте сайт на телефоне
3. Должно появиться предложение установить приложение
4. Или используйте меню браузера → "Добавить на главный экран"

## Проверка PWA:

Используйте Lighthouse в Chrome DevTools:
1. Откройте DevTools (F12)
2. Вкладка "Lighthouse"
3. Выберите "Progressive Web App"
4. Нажмите "Analyze page load"
5. Должен быть высокий балл PWA

## Настройки:

Все настройки PWA находятся в:
- `public/manifest.json` - манифест приложения
- `vite.config.ts` - конфигурация vite-plugin-pwa
- `src/components/InstallPrompt.tsx` - компонент установки

## Важно:

1. **HTTPS обязателен** - PWA работает только по HTTPS (Vercel предоставляет автоматически)
2. **Иконки обязательны** - без иконок PWA не будет работать правильно
3. **Service Worker** - автоматически генерируется при сборке
