# Инструкция по созданию иконок для PWA

## Быстрый способ (онлайн):

1. Зайдите на https://realfavicongenerator.net/
2. Загрузите ваш логотип (минимум 512x512px)
3. Настройте параметры
4. Скачайте пакет иконок
5. Скопируйте `android-chrome-192x192.png` → `public/icon-192.png`
6. Скопируйте `android-chrome-512x512.png` → `public/icon-512.png`

## Альтернативный способ (через браузер):

1. Откройте файл `create-icons.html` в браузере
2. Нажмите кнопку "Скачать иконки"
3. Сохраните файлы в папку `public/`

## Ручной способ:

Используйте любой графический редактор (Photoshop, GIMP, Figma, Canva):

1. Создайте квадратное изображение 192x192px
2. Добавьте ваш логотип или букву "M"
3. Сохраните как `public/icon-192.png`
4. Создайте квадратное изображение 512x512px
5. Добавьте ваш логотип или букву "M"
6. Сохраните как `public/icon-512.png`

## Временное решение:

Для тестирования можно использовать простые цветные квадраты:
- Синий фон (#3b82f6)
- Белая буква "M" по центру
- Скругленные углы (опционально)

## Важно:

- Формат: PNG
- Размеры: точно 192x192 и 512x512 пикселей
- Прозрачность: не обязательна, но желательна
- Фон: должен быть виден на светлом и темном фоне

После создания иконок пересоберите проект:
```bash
npm run build
```
