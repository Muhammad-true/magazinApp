# Инструкция по запуску

## Быстрый старт

1. **Установите зависимости:**
```bash
npm install
```

2. **Создайте файл `.env` в корне проекта:**
```env
VITE_API_URL=http://localhost:8080/api/v1
```
Замените URL на адрес вашего API сервера.

3. **Запустите dev-сервер:**
```bash
npm run dev
```

4. **Откройте браузер:**
Сайт будет доступен по адресу `http://localhost:3000`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`. Их можно загрузить на любой веб-сервер.

## Настройка API

Убедитесь, что ваш API сервер запущен и доступен по адресу, указанному в `.env`.

API должен поддерживать следующие эндпоинты:
- `POST /api/v1/shop-registration/register` - Регистрация магазина
- `GET /api/v1/subscriptions/plans` - Получение планов подписки
- `POST /api/v1/shop-registration/subscribe` - Создание подписки

Подробная документация API находится в файле `API_LICENSE_DOCUMENTATION.md`.

## Структура проекта

- `src/pages/` - Страницы приложения
- `src/components/` - Переиспользуемые компоненты
- `src/services/` - API клиент
- `src/locales/` - Файлы переводов (ru, en, uz)

## Мультиязычность

Сайт поддерживает 3 языка:
- Русский (ru) - по умолчанию
- Английский (en)
- Узбекский (uz)

Переключение языка доступно в навигационной панели.

## Адаптивность

Сайт полностью адаптирован для:
- Десктопов (1200px+)
- Планшетов (768px - 1199px)
- Мобильных устройств (до 767px)

Все изображения автоматически адаптируются под размер экрана.
