## Введение

Данная программа предназначена для управления магазином одежды. Она включает в себя функции кассы, управления товарами, клиентами, отчетности и интеграции с мобильным приложением.

### Основные возможности

- **Касса**: Продажа товаров, работа с чеками, расчет скидок и бонусов
- **Управление товарами**: Добавление, редактирование, управление складом
- **Управление клиентами**: База клиентов, карты лояльности, бонусы
- **Отчеты**: Статистика продаж, аналитика, графики
- **Мобильное приложение**: Интеграция с глобальным сервером для мобильных устройств
- **Обновления**: Автоматическое обновление сервера и приложения

## Начало работы

### Первый запуск

1. **Запустите приложение**
   - При первом запуске вам будет предложено ввести базовый URL API сервера
   - Для других касс (не Касса 1): запустите приложение на Кассе 1, зайдите в админку, откройте раздел "Настройки" и найдите IP-адрес сервера в разделе "Подключение к серверу"

2. **Вход в систему**
   - Введите логин и пароль администратора
   - После успешного входа вы попадете на главный экран

3. **Настройка магазина**
   - Перейдите в раздел "Настройки" в админ-панели
   - Укажите название магазина
   - Настройте другие параметры при необходимости

### Требования

- Оба устройства (сервер и клиент) должны быть подключены к одной Wi-Fi сети
- На сервере должен быть запущен API сервер
- Правильно указан IP-адрес сервера в формате: `http://192.168.1.34:8080/api`

## Админ-панель

Админ-панель доступна только пользователям с правами администратора. В ней доступны следующие разделы:

### Дашборд

Главная страница админ-панели с общей статистикой:

- **График выручки, прибыли и чистого дохода** - визуализация финансовых показателей за сегодня, неделю и месяц
- **Выручка** - общая сумма продаж за период
- **Прибыль** - выручка минус себестоимость
- **Чистый доход** - прибыль минус скидки и бонусы
- **Количество чеков** - число совершенных продаж
- **Средний чек** - средняя сумма покупки
- **Детализация** - себестоимость, скидки, использованные бонусы
- **Активные смены** - количество открытых смен кассиров

#### Пояснение терминов

- **Выручка** - общая сумма всех продаж за период
- **Прибыль** - выручка минус себестоимость товаров
- **Чистый доход** - прибыль минус предоставленные скидки и использованные бонусы

Пример: Товар куплен за 100₽ (себестоимость), продан за 300₽ (цена продажи), предоставлена скидка 50₽, использованы бонусы 20₽.
- Выручка: 300₽
- Прибыль: 300₽ - 100₽ = 200₽
- Чистый доход: 200₽ - 50₽ - 20₽ = 130₽

### Товары

Управление товарами и складом:

- **Список товаров** - просмотр всех товаров с возможностью поиска и фильтрации
- **Добавление товара** - создание нового товара с вариациями (размеры, цвета)
- **Редактирование товара** - изменение информации о товаре
- **Управление складом** - корректировка остатков, добавление/списание товаров
- **Загрузка изображений** - добавление фотографий товаров

### Клиенты

Управление базой клиентов:

- **Список клиентов** - просмотр всех клиентов
- **Поиск клиента** - поиск по имени, телефону, номеру карты
- **Редактирование клиента** - изменение данных клиента, номера карты
- **История покупок** - просмотр истории покупок клиента
- **Управление бонусами** - начисление и списание бонусов

### Отчеты

Детальная аналитика продаж:

- **Продажи по категориям** - статистика продаж по категориям товаров
- **Продажи по цветам** - график и статистика продаж по цветам
- **Топ товаров** - самые продаваемые товары
- **Фильтры по дате** - выбор периода для анализа

### Смены

Управление рабочими сменами:

- **Открытие смены** - начало рабочей смены кассира
- **Закрытие смены** - завершение смены с отчетом
- **История смен** - просмотр всех смен

### Пользователи

Управление пользователями системы:

- **Список пользователей** - все пользователи системы
- **Добавление пользователя** - создание нового пользователя
- **Редактирование пользователя** - изменение данных и прав доступа
- **Роли и права** - настройка прав доступа для разных ролей

### Мобильное приложение

Интеграция с глобальным сервером для мобильных устройств:

- **Вход на глобальный сервер** - авторизация на сервере мобильного приложения
- **Отправка товаров** - загрузка товаров на глобальный сервер для мобильного приложения
- **Выбор категорий** - привязка товаров к категориям глобального сервера
- **Просмотр загруженных товаров** - список товаров на глобальном сервере
- **Удаление товаров** - удаление товаров с глобального сервера
- **Просмотр заказов** - заказы из мобильного приложения

**Важно**: Перед отправкой товаров убедитесь, что у всех вариаций есть фотографии. Товары без фотографий не будут отправлены.

### Обновления

Управление обновлениями системы:

#### Обновления сервера

- **Проверка обновлений** - проверка наличия новых версий сервера
- **Загрузка обновлений** - скачивание и установка обновлений сервера
- **Информация о версии** - текущая и последняя доступная версия

#### Обновления приложения

- **Проверка обновлений** - проверка наличия новых версий приложения
- **Загрузка обновлений** - скачивание обновлений приложения
- **Установка обновлений** - установка обновлений (зависит от платформы)

### Настройки

Общие настройки приложения:

- **Название магазина** - название вашего магазина
- **Подключение к серверу** - настройки подключения к API серверу
- **Другие параметры** - дополнительные настройки системы

## Работа с кассой

### Открытие смены

1. Войдите в систему как кассир
2. Нажмите кнопку "Открыть смену"
3. Укажите начальную сумму в кассе (если требуется)
4. Смена открыта, можно начинать работу

### Продажа товара

1. **Поиск товара**
   - Введите название, артикул или штрихкод товара в поле поиска
   - Или отсканируйте штрихкод сканером

2. **Выбор вариации**
   - Выберите размер и цвет товара
   - Укажите количество

3. **Добавление клиента (опционально)**
   - Нажмите "Выбрать клиента"
   - Найдите клиента по имени, телефону или номеру карты
   - Или отсканируйте QR-код/штрихкод карты клиента

4. **Применение скидок и бонусов**
   - Если у клиента есть скидка, она применится автоматически
   - Можно использовать накопленные бонусы клиента

5. **Оформление чека**
   - Проверьте итоговую сумму
   - Нажмите "Оформить чек"
   - Чек будет сохранен, данные обновятся автоматически

### Закрытие смены

1. Нажмите кнопку "Закрыть смену"
2. Проверьте итоговую сумму в кассе
3. Укажите фактическую сумму (если требуется)
4. Смена будет закрыта, будет сформирован отчет

## Управление товарами

### Добавление товара

1. Перейдите в раздел "Товары" → "Добавить товар"
2. Заполните основную информацию:
   - Название товара
   - Категория
   - Описание (опционально)
3. Добавьте вариации:
   - Размеры (S, M, L, XL и т.д.)
   - Цвета
   - Цена продажи
   - Себестоимость (опционально)
   - Количество на складе
   - Штрихкод (опционально)
   - Инвентарный код (опционально)
4. Загрузите фотографии для каждой вариации
5. Нажмите "Сохранить"

### Редактирование товара

1. Найдите товар в списке
2. Нажмите на товар или кнопку "Редактировать"
3. Измените нужные данные
4. Сохраните изменения

### Управление складом

1. Найдите товар в списке
2. Нажмите "Редактировать вариацию"
3. Измените количество на складе:
   - **Добавить** - увеличить количество
   - **Списать** - уменьшить количество
   - **Установить** - задать точное количество
4. Сохраните изменения

### Загрузка изображений

1. При добавлении или редактировании товара нажмите "Добавить фото"
2. Выберите изображение с компьютера
3. Изображение будет загружено на сервер
4. Можно добавить несколько изображений для одной вариации

## Мобильное приложение

### Вход на глобальный сервер

1. Перейдите в раздел "Мобильное приложение"
2. Введите телефон и пароль для входа на глобальный сервер
3. Нажмите "Войти"
4. После успешного входа станут доступны функции работы с мобильным приложением

### Отправка товаров на глобальный сервер

1. Убедитесь, что вы вошли на глобальный сервер
2. Перейдите на вкладку "Товары"
3. Выберите товары для отправки (отметьте чекбоксы)
4. Для каждого товара выберите категорию на глобальном сервере
5. Нажмите "Отправить выбранные товары"
6. Дождитесь завершения загрузки

**Важно**:
- Товары без фотографий не будут отправлены
- Товары, которые уже были загружены ранее, будут пропущены
- Процесс загрузки может занять некоторое время в зависимости от количества товаров и изображений

### Просмотр загруженных товаров

1. Перейдите на вкладку "Товары с глобального сервера"
2. Просмотрите список всех загруженных товаров
3. Используйте фильтры:
   - По категории
   - По полу (мужской, женский, унисекс)
   - Поиск по названию

### Удаление товаров с глобального сервера

1. Найдите товар в списке "Товары с глобального сервера"
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление
4. Товар будет удален с глобального сервера

### Просмотр заказов

1. Перейдите на вкладку "Заказы"
2. Просмотрите список заказов из мобильного приложения
3. Используйте фильтры по статусу и поиск
4. Можно изменить статус заказа

## Часто задаваемые вопросы

### Как найти IP-адрес сервера?

1. Запустите приложение на Кассе 1 (сервере)
2. Войдите в админку
3. Откройте раздел "Настройки"
4. Найдите IP-адрес в разделе "Подключение к серверу"
5. Скопируйте этот адрес и используйте его на других устройствах

### Почему товары не отправляются на глобальный сервер?

- Убедитесь, что вы вошли на глобальный сервер
- Проверьте, что у всех вариаций товара есть фотографии
- Товары без фотографий не будут отправлены

### Как добавить фотографию к товару?

1. Откройте товар для редактирования
2. Выберите вариацию
3. Нажмите "Добавить фото"
4. Выберите изображение с компьютера
5. Сохраните изменения

### Как работает система бонусов?

- При каждой покупке клиент получает бонусы (если настроено)
- Бонусы можно использовать при следующих покупках
- Администратор может начислять и списывать бонусы вручную

### Как закрыть смену?

1. Нажмите кнопку "Закрыть смену" в интерфейсе кассира
2. Проверьте итоговую сумму
3. Укажите фактическую сумму (если требуется)
4. Подтвердите закрытие смены

### Что делать, если не подключается к серверу?

1. Проверьте, что оба устройства подключены к одной Wi-Fi сети
2. Убедитесь, что сервер запущен на Кассе 1
3. Проверьте правильность IP-адреса
4. Проверьте настройки брандмауэра
5. Попробуйте перезапустить приложение

### Как обновить программу?

1. Перейдите в раздел "Обновления"
2. Выберите вкладку "Обновления приложения" или "Обновления сервера"
3. Нажмите "Проверить обновления"
4. Если доступна новая версия, следуйте инструкциям на экране
